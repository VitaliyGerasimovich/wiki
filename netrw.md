# Netrw

- Common commands:
  - `-` => go to up directory
  - `gb` => go to previous marked directory
  - `gn` => go to top tree below cursor
  - `D` => attempt to remove
  - `mx` => shell command for marked files
  - `u` => go to previous directory
  - `R` => rename
  - `i` => information/tree
  - `s` => sort
  - `r` => reverse sort
